JavaScript Expense Tracker
Overview

This project is a simple Expense Tracker made with HTML, CSS, and JavaScript. It lets users add expenses, choose a category, select USD or PHP, and see their total spending. Users can also delete expenses, keep their data saved with localStorage, and view their spending by category using a chart.

I made this project to practice JavaScript skills like functions, arrays, objects, DOM manipulation, localStorage, recursion, Chart.js, and error handling.
